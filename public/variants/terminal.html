# about

I build secure LLM systems for the enterprise — retrieval assistants that reason over internal documentation and code, guardrail and security analysis for LLM applications, and tooling that wires LLMs into CI/CD pipelines.

That practice is grounded in a Ph.D. in nonconvex optimization, graph theory, and multi-armed bandits. # the same math that proves an algorithm converges also tells you where a system will fail.
